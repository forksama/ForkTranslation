# 立绘处理流程

这份文档记录 `怪文书素材/1.立绘` 下角色立绘的批处理流程。脚本统一放在 [scripts/standing_images](../scripts/standing_images) 目录里；下一次继续维护时，建议先读 [scripts/README.md](../scripts/README.md) 和 [scripts/standing_images/README.md](../scripts/standing_images/README.md)。

脚本默认根目录已经指向：

```text
C:\Repositories\ForkTranslation\怪文书素材\1.立绘
```

如果以后想先在临时目录试跑，可以给脚本额外传 `--root "C:\...\立绘tmp"`。

## 目录约定

每个角色目录下使用这些子目录：

- `七分像`：七分身源图。
- `半身像`：最终半身图目录。
- `半身像-2`：审核用临时半身图目录。

## 第一步：裁掉透明边

脚本：[scripts/standing_images/trim_alpha_edges.py](../scripts/standing_images/trim_alpha_edges.py)

用途：

- 遍历每个角色目录下的 `半身像` 和 `七分像`。
- 对 `.png` 图片按 alpha 通道计算非完全透明像素的最小外接矩形。
- 裁掉四周多余的全透明区域，图片内部原有透明通道会保留。
- 默认跳过 `_` 开头的目录，以及名称包含 `_裁剪` 的辅助目录。
- 默认不处理 `半身像-2`。

先预览，不覆盖任何文件：

```powershell
python scripts/standing_images/trim_alpha_edges.py
```

确认后覆盖原图：

```powershell
python scripts/standing_images/trim_alpha_edges.py --execute
```

只处理某个角色目录：

```powershell
python scripts/standing_images/trim_alpha_edges.py --character-dir "C:\Repositories\ForkTranslation\怪文书素材\1.立绘\1-咲季立绘.1"
```

覆盖前额外备份到指定目录：

```powershell
python scripts/standing_images/trim_alpha_edges.py --execute --backup-dir "C:\Repositories\ForkTranslation\怪文书素材\立绘_alpha_trim_backup"
```

## 第二步：从七分像生成半身像-2

脚本：[scripts/standing_images/generate_half_body_2.py](../scripts/standing_images/generate_half_body_2.py)

用途：

- 在某个角色目录的 `半身像-2` 中放一个审核示例图。
- 脚本找到 `七分像` 中同名图片，用示例高度 / 七分像高度推导“保留上方多少比例”。
- 将该角色 `七分像` 中所有 `.png` 按同一高度比例从下方裁掉。
- 写入前再按 alpha 通道裁掉四周全透明边。
- 输出到同角色的 `半身像-2`，文件名保持和 `七分像` 一致。

先预览，不写入文件：

```powershell
python scripts/standing_images/generate_half_body_2.py --character-dir "C:\Repositories\ForkTranslation\怪文书素材\1.立绘\1-咲季立绘.1"
```

确认后写入 `半身像-2`：

```powershell
python scripts/standing_images/generate_half_body_2.py --character-dir "C:\Repositories\ForkTranslation\怪文书素材\1.立绘\1-咲季立绘.1" --execute
```

如果 `半身像-2` 里有多个可用示例，需要显式指定：

```powershell
python scripts/standing_images/generate_half_body_2.py --character-dir "C:\Repositories\ForkTranslation\怪文书素材\1.立绘\1-咲季立绘.1" --example 开心.png
```

## 第三步：审核后回归半身像

脚本：[scripts/standing_images/promote_half_body_2.py](../scripts/standing_images/promote_half_body_2.py)

用途：

- 将审核通过的 `半身像-2` 图片复制到 `半身像`。
- 如果 `半身像` 中已存在同名文件，则跳过，不覆盖。
- 处理完成后删除对应的 `半身像-2` 源文件，保留空目录。

先预览，不写入也不删除：

```powershell
python scripts/standing_images/promote_half_body_2.py --character-dir "C:\Repositories\ForkTranslation\怪文书素材\1.立绘\1-咲季立绘.1"
```

审核确认后执行：

```powershell
python scripts/standing_images/promote_half_body_2.py --character-dir "C:\Repositories\ForkTranslation\怪文书素材\1.立绘\1-咲季立绘.1" --execute
```

全量执行：

```powershell
python scripts/standing_images/promote_half_body_2.py --execute
```

执行时如果想保留 `半身像-2` 删除前备份：

```powershell
python scripts/standing_images/promote_half_body_2.py --execute --backup-dir "C:\Repositories\ForkTranslation\怪文书素材\立绘_promote_backup"
```

## 推荐顺序

1. 准备或更新各角色的 `半身像`、`七分像`、`半身像-2` 目录。
2. 跑第一步，裁掉 `半身像` 和 `七分像` 的外圈透明边。
3. 在每个角色的 `半身像-2` 中放一个同名示例图。
4. 跑第二步，生成全部审核用 `半身像-2`。
5. 人工审核 `半身像-2`。
6. 审核通过后跑第三步，把 `半身像-2` 合并回 `半身像` 并清空源目录。

每个脚本默认都是 dry-run；只有加 `--execute` 才会改文件。
