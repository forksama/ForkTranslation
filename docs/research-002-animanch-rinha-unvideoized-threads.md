# 调研档案 002：贺阳燐羽中心串的未视频化作品调查

- 档案编号：Research-002
- 建档日期：2026-07-12
- 状态：已完成初次站内复查；2026-07-15 复查时修正《あの賀陽燐羽さんですか？》《賀陽継と賀陽燐羽》的视频化状态
- 主题：あにまん掲示板中以贺阳燐羽为中心的精品串、未视频化作品与台词体SS

本文不使用此前依赖视频索引所得的调查结论，而是通过あにまん掲示板的正规搜索入口重新调查所得。

## 结论

- 在已经确认的范围内，符合“单串150楼以上、以燐羽为中心、非安价／非安科”三个条件的有力作品均已被视频化。
- 未视频化作品中最长的有力候选是[《賀陽燐羽の日記》](https://bbs.animanch.com/board/4502599/)82楼，加上[《賀陽燐羽の日記2》](https://bbs.animanch.com/board/4588899/)105楼，**系列合计187楼，正文约4.8万字**。
- 在以“人物名＋台词”推进的非安价、非安科作品中，最值得推荐的未视频化作品是[《賀陽燐羽親愛度コミュ妄想SS》](https://bbs.animanch.com/board/6830978/)。它虽然只有65楼，但收录了完整10话、约2.5万字，实际篇幅远高于楼数给人的印象。
- 本文中的“未视频化”最初指截至2026-07-12，通过标题、标题变体和board编号，在YouTube、Bilibili、Yutura、ブィレーダー及Digital Creators中均未找到视频。2026-07-15 复查发现《あの賀陽燐羽さんですか？》《賀陽継と賀陽燐羽》已在 YouTube 视频化，因此本文已将它们从未视频化候选中移出。未发现视频不能排除已经删除、私人或不公开的视频。

## 调研方法

使用了あにまん掲示板的三个正规入口：

- 现行串：`https://bbs.animanch.com/search/关键词`
- 过去串：`https://bbs.animanch.com/search2/关键词`
- 回复正文β：`https://bbs.animanch.com/searchRes/关键词`

搜索词包括`賀陽燐羽`、`燐羽`、`りんは`、`燐羽様`、`SyngUp`、`りんちゃん`，以及主要SS标题和关系词。随后直接读取每个候选串的正文，检查楼数、正文长度、会话符号和`dice`、`安価`、`あんこ`的使用情况。

视频化判断则通过完整标题、缩略标题、视频改题和board编号进行反向检索。

# 1. 最优先：未视频化精品串

## 1.1 賀陽燐羽の日記／賀陽燐羽の日記2

- [第一部：board/4502599](https://bbs.animanch.com/board/4502599/)——82楼，正文约1.65万字
- [第二部：board/4588899](https://bbs.animanch.com/board/4588899/)——105楼，正文约3.15万字

- 视频化：**未发现**
- 安价／安科：无
- 中心人物：贺阳燐羽；极月一方的P和白草四音也很重要
- 形式：燐羽第一人称日记，并非纯粹的`燐羽「」`形式
- 完成度：高，是从第一部延续到第二部的连续作品

内容概要：以燐羽的日记描写她转入极月、签订隐退演唱会契约、与极月P建立关系、重新恢复偶像能力、和白草四音开展组合活动以及制作个人曲目的过程。作为利用官方设定构筑的架空路线，篇幅和可读性均较高。

推荐度：**最高级**。它是本次未视频化候选中，篇幅与故事性结合得最好的一部。
