# 学马仕翻译决策速查

本文件只保留**最终怎么译**。理由、来源、调研过程不放入默认翻译上下文；需要追溯时查 `docs/` 归档或 git 历史。

## 全局裁决

| 原文 / 场景 | 译法 | 禁用 / 注意 |
|---|---|---|
| `学園アイドルマスター` | 学园偶像大师 | `学马仕` 只作项目内部简称，不作为正式译名 |
| `プロデューサー` / `P`（角色称呼或叙述身份） | 学 P | 对话正文中若角色固定称呼为「制作人」可按口吻使用；字幕 role 仍用 `学 P` |
| 担当偶像中文名 | 以 `glossary.csv` 为准 | 不在单串内另起译名 |
| `SyngUp!` | SyngUp! | 不译「震升!」「声扬!」等中文化名 |
| `Re;IRIS` | Re;IRIS | 保留分号 |
| `H.I.F` / `N.I.A` | H.I.F / N.I.A | 保留点号和英文缩写 |
| `一番星`（称号） | 一番星 | 不译「最亮的星」「第一颗星」 |

## 角色名裁决

| 原文 | 译法 | 禁用 / 注意 |
|---|---|---|
| `藤田ことね` | 藤田琴音 | 不用「藤田言音」 |
| `ことねちゃん` 等派生称呼 | 以「琴音」为基础处理 | 是否保留 `ちゃん` 按 `style-guide.md` 的称呼规则 |

## 使用边界

- 本文件只覆盖**反复出现且容易译乱**的裁决。
- 角色口吻看 `characters.md` / `character-profiles/*.md`。
- 人物关系看本串相关的 `relationships.md` 条目。
- 剧情背景只在原文明确提到事件、系统或专名时查 `common-background.md`。
- 社区绰号、二创流派、黑话只在原文实际出现时查 `community-expressions.md`，不要默认加载。
