# 学马仕背景补充速查

本文件不是默认翻译上下文。只有原文明确提到剧情事件、系统词、学园结构或世界线约束时再查。角色口吻优先看 `characters.md` / `character-profiles/*.md`，人物关系优先看本串相关的 `relationships.md` 条目。

## 默认只记住

- **学P×偶像是并行世界线**：一个培育周次只担当一位偶像。翻译具体串时先确认当前担当偶像；不要把其他担当线的私人关系带进来。
- **组合层关系可共通，私人关系不共通**：如 Re;IRIS 结成可作为组合线背景；但“某偶像与学P的亲密度”只在该偶像担当线成立。
- **N.I.A 在当前 1 年生线中早于 H.I.F**。不要把当前线 H.I.F 排到 N.I.A 前。
- **H.I.F 是反复举办的制度名**：賀陽継那届 H.I.F 属前史，当前 1 年生线 H.I.F 是另一届，不要混成同一事件。

## 系统词

系统词具体译名以 `glossary.csv` 为准。这里只列容易误解的判断。

| 原文 / 概念 | 译法 / 注意 |
|---|---|
| プロデュース | 培育；不译“制作/生产” |
| 担当 | 担当；不解释成“负责的偶像” |
| Sense / Logic / Anomaly | 感性 / 理性 / 非凡 |
| ロジック | 理性；不用“逻辑”作主译 |
| アノマリー | 非凡；不用“异常/阿诺马丽” |
| Vo / Da / Vi | 缩写保留；全称为歌唱 / 舞蹈 / 视觉表现 |
| P カード / S カード | P 卡 / S 卡 |
| 好感度 | 系统数值名 |
| 親愛度コミュ | 剧情类型名，可译“亲爱度剧情” |
| Lv | 保留字母，如“好感度 Lv1” |
| コミュ | 剧情；不译“交流/沟通” |
| 才能 | 保留“才能” |
| 才能を見抜く | 看出才能 |

## 系列差异

- 主系列通常说“制作人”，学马仕中作为角色称呼固定用“学P”。
- 学马仕培育有 Sense / Logic / Anomaly 三分支。
- 学马仕的 P × 偶像私人关系按并行世界线处理，不按同一时空群像处理。
