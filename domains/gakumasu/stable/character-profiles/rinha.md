## 2. 官方形象与性格核

**表面**：偽悪的、语气冷淡挑衅，被社区归为"グレた（叛逆化）"偶像。看客易读作反派。

**深层**：**温柔正直，正直到不知如何自处**。日文考察文（[note @bs](https://note.com/bsgang/n/n8e93566e7541)）总结为"あまりにも優しくて、真っ直ぐすぎる"擦れ方——正因太温柔太直才走岔了路。她的核心动机是"为了不再伤害任何人，压抑自身的冲动性、试图保持冷静形象"。

**性格转折**：中等部 SyngUp! 时期性格开朗，与美鈴之间有被称为"悪癖"的互动（[ニコニコ大百科](https://dic.nicovideo.jp/a/賀陽燐羽)）。因**姐姐継的引退**与 **SyngUp! 的解散**双重打击，从"純粋だった性格"变为"大人びた、どこか脆さや歪さ、諦めをも感じる"（[X @HKRS_gakumas](https://x.com/HKRS_gakumas/status/2056290159105368501)）。

**弱点**：**被他人憧憬**。SyngUp! 时期手毬 / 美鈴憧憬着自己这一事实被戳破时她会显著动摇——形容为"脅迫状の100倍キツい"。这是她与其他"傲沉型"角色最不同的心理防线。

**能力反差**：**歌唱实力真材实料且不吝努力**，被手毬评价为"声や実力共に姉である賀陽継の完全コピー"。但本性怠惰、贪吃，被评为"筋肉より脂肪で構築された"身体（[note @oimo_pi](https://note.com/oimo_pi/n/nad6f003abdcb)）。

**与姐姐的关系**：并非单纯憧憬或竞争。可能因姐姐引退违背了对妹妹的某种"约定"（[note @aporo_310](https://note.com/aporo_310_poke/n/n50728e9541d3)），导致燐羽性格突变。但从美鈴路线可窥"燐羽から継への感情が思ったほど悪くなさそう"——**憧憬为底色，混杂被违约的伤痕，非单纯负面**。

## 3. 说话风格与口癖

- **一人称**：以「あたし」为常；防御姿态或激烈情绪时切「私」（书面感增强）。
- **对陌生人 / 制作人的第一反应**：偏挑衅型开场——"はぁ？"、"何なの？"、"うざい"，语气短促、断句密。
- **对 SyngUp! 旧成员**（手毬 / 美鈴）：撑不住偽悪，句尾会突然放软，甚至冒出照顾人 / 后悔的一面。
- **感情失衡的招牌反应**：「お、おバカ！」——被戳到痛处或看到手毬哭时的失态口吻，是社区反复引用的名场面（[あにまん 4369910](https://bbs.animanch.com/board/4369910/)）。
- **不使用**日式撒娇符号（♪ / ♡）作为常态口癖——出现基本都是二创加料。
- **语气节奏公式**：**冷 → 冷 → 一瞬破** 的三拍模式。前两拍维持冷淡短句，第三拍留出破防的语速转变。这个节奏是她与其他"傲沉"角色（如手毬）最本质的区别。

## 6. 二创偏差与翻译边界

只列 2 个最容易导致翻译走偏的方向。**引用二创作品时必须显式标注"来自二创，非官方"**。

### 6.1 「Pラブ / 依存系」倾向（あにまん 特有）

- **偏差**：把 "SyngUp!" 解读为 Sin+ing / 七宗罪，并把燐羽推向 P ラブ、依存或情欲暗示。
- **翻译边界**：不因二创倾向自动把燐羽对 P 的敬意 / 敌意翻译成"依恋"或"情欲暗示"。**原文没写就是没写**——燐羽的能量主要指向姐姐、SyngUp! 旧友、初星阵营，不是 P。

### 6.2 「病态 / 幼态化 / 心中依附」倾向

- **偏差**：把 §2 中"温柔本心"或"諦めの歪さ"推向幼态化、心中依附或病态爆发。
- **翻译边界**：官方处理是"沉淀后的成熟"，**不是"病态爆发"或"幼态撒娇"**。原文"眠れない"就译"睡不着"，不译"根本没法活"。不要主动追加撒娇符号（如「♡」）或把台词幼态化；但符合中文语境的「呢」「吧」「啊」「啦」等语气词可按 `style-guide.md` 使用。译文强度**不领先原文半步**。

### 6.3 用户审校偏好：P ラブ / 照顾生活向短篇

- **适用范围**：仅限あにまん二创中已经明确写出燐羽主动贴近、照顾生活、占有欲或 `支配` 气味的 P ラブ / 通い妻式短篇；不用于官方剧情或普通对话。
- **口吻方向**：比直译稿更口语、更像中文里会说出口的挑衅和亲近台词。优先用短促判断句、反问句和轻微命令感，例如「这算是出轨吧？」「想着我以外的女人是什么意思？」「就老老实实沉浸在我的照顾里吧」。
- **强势表达**：原文已有威胁、支配、占有欲时，可以用更利落的中文呈现，不必保留缓冲词。例如 `殺すから` 可在该类二创里译成「就杀了你」；`簡単にあなたを支配出来そう` 可译「支配你还不是轻而易举吗」。
- **照顾语气**：`お世話` 在该类短篇里不只是不带感情的“照顾”，也带主动侵入生活的亲密感。可译作「把你照顾得无微不至」「沉浸在我的照顾里」「被我的照顾填得更满一些」等，但不得新增原文没有的具体行为。
- **旁白语感**：旁白可比普通审校版更顺口，少用“突然/转眼间”等直译连接词；优先「这一瞬」「没一会儿」「向家的方向走去」这类更自然的叙述。

## 7. 翻译口吻要点（硬约束）

1. **保留短句结构**——她的短句和断句本身承载防御姿态，不拼长、不强断。
2. **一人称**：「あたし」译"我"，靠句式保留 casual；「私」出现在防御姿态时可用更书面的句式承接。
3. **语气词按中文语境使用**：句尾 `〜だけど / 〜じゃない / 〜わけ` 不机械对应，但可按她当下的冷淡、挑衅、认命或破防程度译出自然的"啦""吧""啊""呢"等；重点是别把燐羽翻成幼态撒娇。
4. **破防瞬间加节奏**：「お、おバカ！」译"你、你这笨蛋！"，不平滑为"真是笨蛋"。
5. **面对姐姐 / 旧友时**允许更完整的句子和更长的犹豫——这是她的"未愈合区"，是重要角色深度锚点。
