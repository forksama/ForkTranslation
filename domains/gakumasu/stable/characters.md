# 角色概要（人物地图）

- 建档日期：2026-07-13
- 状态：首版
- 上游规格：[Research-005 §4](../../../docs/research-005-gakumas-stable-knowledge-first-version-spec.md#4-charactersmd-首版规格)

本文档为强 agent 与人工审校提供**全项目人物地图**。每位角色只写"直接影响翻译判断"的最小信息：身份、说话风格、典型台词、主要关系对象。详细性格分析、二创偏差、剧情节点见 `character-profiles/*.md`（若有）。

## 使用规则

- **人名与所属组合**统一按 [`glossary.csv`](glossary.csv)；此处不重复登记译名。
- **人物关系的具体描述**放在 [`relationships.md`](relationships.md)；此处只列出"主要关系对象名"清单。
- **说话风格、口癖、典型台词**：主源为学マス官方角色页与日文百科（Pixiv/ニコニコ）、日文考察类 Note 文章；中文百科（萌娘百科等）作为交叉参照。
- **翻译最易错判点** 首版留空，靠后续实际翻译过程中反推回填。
- 独立 profile 列在【是否有独立 profile】字段；无 profile 表示"信息以本文件为准"。

## 说话风格通用注意事项

- 学マス 1 年生（15 岁）与 3 年生（17 岁）在敬语强度和口气上有明显差别，同龄同班同学之间基本不使用敬体。
- 每位角色**一人称、语尾、口癖**尽量按下方登记；无明确记录的角色首版留白，不脑补。
- 中日字形相同的名字（如「集中」「好印象」）在译文中保留字形。
- 角色对制作人的称呼请见 [style-guide.md §3.1](style-guide.md#31-p-的称呼)：一律用 `学P` 作为 C 的 role，正文可用「制作人」或角色的固定称呼。
- **学 P 与各偶像的关系是并行世界线**——一个 プロデュース 周次只培育一位偶像。以下各角色"主要关系对象"里列出的"学 P"是指**该角色担当世界线**下的关系，**不同角色的学 P 关系不同时成立**。翻译具体串时按其"当前担当偶像"选相应关系。详见 [`relationships.md §9`](relationships.md#九制作人学-p-关系)。
